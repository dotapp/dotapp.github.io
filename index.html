dotapp

OS X application bundle wrappers

A collection of OS X application bundle wrappers.

Usage

Recommended:

  1. Install Homebrew Cask
  2. Tap homebrew-dotapp

Manual:

  1. Pick a dotapp from the list
  2. Clone into your /Applications directory
